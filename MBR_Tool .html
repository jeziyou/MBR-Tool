四、清洗维护

三菱化学PVDF材质膜有三种化学清洗方法: 逆通液清洗(维护清洗)、在线清洗(恢复清洗)、系统外清洗(浸泡清洗)

使用药液: NaClO (去除有机物和生物污堵) | 酸 (去除无机物污堵)

清洗方式用量 (L/m²膜)浓度 (mg/L)备注
维护清洗 (低)2500每周1次, NaClO溶液, 30~60分钟/次
恢复清洗 (高)23000每3个月1次或压差>15kPa, NaClO溶液
浸没清洗-3000在线清洗后压差不能恢复时, 6~24小时/次
酸清洗-1%~2%根据实际情况调整, 参照膜使用手册
禁止使用氢氧化钠 (NaOH) 清洗三菱膜, 会导致膜老化脱皮!

五、故障处理

症状原因对策
压差上升无曝气 (风机故障)确认风机是否异常停止/故障/曝气不均匀, 风机与泵联动停抽吸
压差上升MLSS过高排泥至3000~12000mg/L范围, 控制粘度<30cps
压差上升油类流入设置油水分离, 矿物油禁止流入
压差上升曝气不足检查分流, 确保额定风量, 膜供气优先
压差上升无机物 (Fe/Mn/Ca/Si)预处理去除, 酸清洗 (草酸/柠檬酸/盐酸)
压差上升凝聚剂过量氯化第二铁6~18mgFe/L, 聚氯化铝12~24mg/L, 需试验确定
氮超标DO不足好氧区DO保持1~2mg/L, 缺氧区<0.5
氮超标pH异常维持pH 6~8, pH<6时添加碱度
产水浊度>1NTU管道泄漏/膜破损停止产水, 检查管道密封和膜完整性, 进行清洗
产水浊度>1NTU集水部破损/L集水管脱落检查管道连接, 更换O形圈或集水管
产水含气泡连接部漏气/管道空气积存检查管道密封, 排除积存空气
抽吸压力不稳定过滤水线空气未排尽充分脱气, 抽空气时压力不稳定属正常
过滤水不出膜干燥膜片需重新亲水化处理后再安装
鼓风机压力上升曝气管堵塞清洗曝气管, FP曝气管需取出清洗
清洗空气偏差/不足曝气管堵塞/风机故障确认风量, 清洗或更换曝气管
透水率下降膜污染调整运行参数, 增加MC频率

六、检漏与安装

检漏步骤:

  1. 水槽满水
  2. 集水管一侧堵头 / 一侧接气
  3. 静置10min
  4. 加压28kPa找漏点
  5. 环氧树脂修补
用空气检漏 (禁止水泵) | 起吊角度 ≤ 60° | 膜组件器中心距 2200mm | 曝气管距膜底 ≥ 200mm | 顶部距水面 > 500mm

七、安全与控制

  • 风机故障必须联动停抽吸泵, 防止污泥固化
  • 泵尽量装液位下方, 设破虹吸装置, 吸程不宜 < 6m
  • 启动顺序: 搅拌机 → 循环泵 → 风机 → 抽吸泵
  • 膜池与产水系统联动: 污泥回流泵全停时膜池不能产水
  • 膜池曝气故障 / 风量过低时禁止连续产水
  • 进水在线检测: 流量计 / pH / DO / COD / 氨氮 / 总氮 / 总磷 / 温度 / MLSS
  • 出水检测: COD / BOD5 / 氨氮 / 总氮 / 总磷 / 浊度
  • 消泡剂: 仅限高级酒精类或醚类/酯类, 严禁硅类消泡剂 (吸附膜面导致压差上升且无法药洗)
  • 膜片取出后12小时内须浸泡一次保持湿润, 避免干燥; 干燥后需重新亲水化处理
  • 原则上不需要反洗 (backwash), 按使用说明书运行即可
  • 稀释水泵吐出压建议 30kPa 以上, 药液注入速度为过滤通量的 1/6 左右
  • 脱氮槽最低滞留时间建议确保 1 小时以上
  • MLVSS 通常为 MLSS 的 70~80%
  • SV30 测量: MBR污泥约 10000mg/L, 需稀释4倍后测定
